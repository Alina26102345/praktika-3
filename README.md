# Система учета заявок на ремонт бытовой техники

## Описание проекта

Программный модуль для автоматизации учета заявок на ремонт бытовой техники
в сервисном центре. Разработан в рамках учебной практики.

## Функциональные возможности

### Основные функции:
1. **Управление заявками:**
   - Создание новых заявок
   - Просмотр и редактирование существующих
   - Изменение статусов
   - Назначение мастеров

2. **Поиск и фильтрация:**
   - Поиск по номеру, клиенту, телефону
   - Фильтрация по статусу
   - Сортировка по дате

3. **Статистика и отчетность:**
   - Расчет среднего времени ремонта
   - Статистика по типам устройств
   - Распределение по статусам
   - Отчеты по производительности

4. **Дополнительные функции:**
   - Комментарии и история изменений
   - Учет запчастей
   - Уведомления о смене статусов

## Требования к системе

### Программные требования:
- Python 3.8 или выше
- SQLite3 (входит в стандартную библиотеку)
- Tkinter (для графического интерфейса)

### Аппаратные требования:
- Процессор: 1 ГГц или выше
- Память: 512 МБ ОЗУ
- Дисковое пространство: 50 МБ
